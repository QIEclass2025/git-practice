본 프로젝트는 Python의 Tkinter 라이브러리를 사용하여 사용자 순발력을 측정하는 미니 게임입니다. 단순한 단발성 테스트가 아닌, 5회 평균 측정 시스템을 도입하여 신뢰도를 높였으며, 최종 결과에 따라 PokeAPI에서 무작위 포켓몬을 추천하는 재미 요소를 추가했습니다.

목표 : 가장 빠른 평균 반응 속도를 측정하여 '신' 등급에 도전하고, 그에 어울리는 포켓몬을 추천받는 것입니다.

대기 (Standby): 게임 시작 또는 재도전 시 파란색 원이 뜹니다. 화면 중앙의 파란색 원을 클릭하여 테스트를 시작할 준비를 합니다.

측정 시작 (Wait): 원이 빨간색으로 변하며 무작위 시간(2초~7초) 동안 대기합니다. 이 상태에서 클릭하면 '너무 빨리 클릭'으로 기록됩니다.

반응 (Reaction): 원이 초록색으로 바뀌는 순간, 최대한 빨리 클릭합니다.

피드백: 클릭 성공 여부와 반응 시간이 화면에 표시되며, 1.5초 후 원이 사라집니다. (이 1.5초 동안 실수로 클릭해도 카운트되지 않습니다.)

재시행: 이 과정이 총 5회 반복됩니다.

최종 결과: 5회의 성공 기록(Too Early 제외) 평균을 계산하여 최종 랭킹과 함께 '당신의 반응속도로는 포켓몬 **[이름]**이 어울려요!'라는 멘트와 포켓몬 이미지가 표시됩니다.
